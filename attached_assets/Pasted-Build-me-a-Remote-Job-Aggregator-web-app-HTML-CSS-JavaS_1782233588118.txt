Build me a Remote Job Aggregator web app (HTML + CSS + JavaScript, 
single file) with the following features:

---

DATA SOURCES (Public APIs — no auth required):
- RemoteOK API: https://remoteok.com/api
- Remotive API: https://remotive.com/api/remote-jobs
Fetch from both APIs and merge results, remove duplicates by job title + company name.

---

TIME FILTER (Most Important Feature):
Add a prominent filter bar at the top with these buttons:
- "Last 24 Hours"
- "Last 36 Hours"  
- "Last 48 Hours"
- "Last 7 Days"
- "All Jobs"
Default view should be "Last 48 Hours" on load.
Show a live count badge next to each button like: "Last 24H (23 jobs)"
This count should update dynamically as other filters are applied.

---

THEME SWITCHER:
Add a toggle in the top-right corner to switch between:
- Light Mode
- Dark Mode
- Midnight Blue (dark blue theme)
Save the user's theme preference in localStorage so it persists on refresh.

---

JOB CARD DESIGN:
Each job card should show:
- Company logo (if available, else placeholder initials avatar)
- Job Title (bold)
- Company Name
- "NEW" green badge if posted within last 6 hours
- Time posted (e.g., "3 hours ago", "1 day ago")
- Salary range (if available) — show "Salary not listed" in grey if missing
- Tags: tech stack / skills (e.g., React, Python, Node.js)
- Location restrictions — if listing contains "US Only", "EU Only", 
  "UK Only", "Right to work", "Must be in" — show a bold RED WARNING 
  badge that says "⚠️ Location Restricted"
- A green badge "✅ Truly Remote" if no location restriction detected
- "Apply Now" button that opens the job URL in a new tab
- "Save Job" bookmark icon that saves to a local list

---

FILTERS PANEL (Left sidebar or top filter row):
1. Experience Level:
   - Fresher / Entry Level (0-2 years)
   - Mid Level (2-5 years)
   - Senior (5+ years)
   - Any
   Detection: scan title and description for keywords like 
   "junior", "entry", "senior", "lead", "0-2 years" etc.

2. Salary Filter:
   - Any
   - $0 - $50k
   - $50k - $100k
   - $100k+
   - Only show jobs with salary listed

3. Job Category:
   - All
   - Engineering
   - Design
   - Marketing
   - Product
   - Data / ML
   - Customer Support
   - Writing / Content
   - DevOps / Infra

4. Search Bar:
   - Real-time search across job title, company name, and tags

---

SAVED JOBS TAB:
Add a "Saved Jobs" tab/section where bookmarked jobs are stored 
using localStorage. User can remove saved jobs. Show count of 
saved jobs in the tab label.

---

STATS BAR:
Show a small stats row below the header:
- Total jobs fetched
- Jobs in last 24H
- Jobs in last 48H
- Truly Remote count
- Location Restricted count

---

SORTING OPTIONS:
Dropdown to sort by:
- Newest First (default)
- Oldest First
- Salary: High to Low
- Salary: Low to High

---

LOADING & ERROR STATES:
- Show a skeleton loader animation while fetching
- If an API fails, show a small warning but still load the other source
- Add a "Refresh Jobs" button that re-fetches all data
- Show "Last updated: X minutes ago" timestamp

---

GENERAL UI REQUIREMENTS:
- Fully responsive (mobile + desktop)
- Clean, modern design — use Inter or Poppins font from Google Fonts
- Smooth animations on filter changes (fade/slide)
- No external frameworks — pure HTML, CSS, JS only
- All in a single index.html file
- Add a small footer: "Data sourced from RemoteOK & Remotive • 
  Updates every 30 min"
- Auto-refresh data every 30 minutes silently in background

---

Make the app feel premium and fast. Prioritize clean UX over 
heavy features. The target users are college graduates and 
developers with 1-3 years of experience looking for genuine 
remote opportunities.
